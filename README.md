# Automação residencial 2.0 - feito com arduino

## Sobre
```
Projeto feito em 2018 para a conclusão do curso de arduino no Senac - PI, e refatorado em 2019.
O projeto tinha por finalidade trabalhar com as novas tecnologia aprendidas ao longo do curso e 
trazer para o dia a dia assim surgindo a ideia da automação residencial mais simples e barata e 
totalmente customizavél por quem fosse utilizar.
Foram utilizados além da tecnologia do Arduino também tecnologia do dia a dia como
- Celular
- Bluetooth
- Assistente do google
```

## Requisitos do projeto
```
- Arduino Mega/Uno
- Bluetooth 4.0/5.0
- Relé 4 Canais
- Sensor de Luminosidade
- Leds
- 2 Lampadas 220v 
- Placa organizadora
- LCD 16x2 com I2C
- Smartphone Android
- Sensor RFID
- Cartão RFID
- TAG RFID
- Apps para integração com o google assistente
    - Tasker
    - AutoVoice
    - AutoApps
    - Assistent google
```
